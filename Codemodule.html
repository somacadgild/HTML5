    What Are Front End Technologies



    For building a web application you need to focus on front-end layer for preparing the user interface. Generally, you have to write a code which will provide the look and feel of the application. It is usually visible to the user in the form of an interface inviting to interact with the user. Front end technologies help us to interact with the user and define the data in the proper way. HTML5: HTML (or HyperText Markup Language) is responsible for creating a markup of your website and letting the browser read it and display it correctly.HTML is not a programming language however it is treated as the markup based language which help us to prepare the website layouts. The version is now HTML5 which is industry recommended. HTML API are in demand example Web Storage, Canvas, SVG, Drag and Drop, Geolocation.
